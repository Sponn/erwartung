# AccumulatorValue
cim:AccumulatorValue

AccumulatorValue represents an accumulated (counted) MeasurementValue.

*Inherits from: MeasurementValue*

## Validations

### value

The value to supervise. The value is positive.

- **Minimum Value**: 0.0
